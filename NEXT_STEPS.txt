================================================================================
REFERÊNCIAS RÁPIDAS:
================================================================================

Conversão Básica:
  Tkinter Frame      →  PyQt6 QWidget
  Tkinter Toplevel   →  PyQt6 QDialog
  Tkinter Label      →  PyQt6 QLabel
  Tkinter Button     →  PyQt6 QPushButton
  Tkinter Entry      →  PyQt6 QLineEdit

Layouts:
  .pack()            →  layout.addWidget()
  .grid()            →  layout.addWidget(row, col)
  .geometry()        →  .setGeometry(x, y, w, h)

Variáveis:
  StringVar()        →  str (variável normal)
  IntVar()           →  int (variável normal)
  BooleanVar()       →  bool (variável normal)

Eventos:
  .bind()            →  .connect() (com signals/slots)
  messagebox.show()  →  QMessageBox.information()
